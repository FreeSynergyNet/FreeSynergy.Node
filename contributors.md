# Contributing to FreeSynergy.Node

Thank you for your interest in FreeSynergy.Node!

## Current Status: Contributions Not Yet Accepted

FreeSynergy.Node is in its initial build phase. During this period,
we are **not accepting external contributions** (pull requests, patches,
or code submissions).

### Why?

We are developing a custom license that reflects the project's core values:
freedom, decentralization, and voluntary cooperation. Until this license is
finalized, accepting contributions would create unclear ownership situations
that could cause problems for everyone involved – contributors and
maintainers alike.

We want to get this right, so that when we open the project for
contributions, the terms are fair, clear, and transparent from day one.

### What You Can Do Now

- **Use it.** Download, install, run, modify – the MIT license allows all of this.
- **Share it.** Tell others about FreeSynergy.Node.
- **Report issues.** Bug reports and ideas are welcome via GitHub Issues.
- **Star the repo.** It helps visibility.

### When Will Contributions Open?

Once the project license and a Contributor License Agreement (CLA) are
in place, we will open FreeSynergy.Node for contributions. This file
will be updated with full contribution guidelines at that time.

Questions? Open an issue or reach out via [FreeSynergy.Net](https://freesynergy.net).
